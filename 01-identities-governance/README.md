# Module 1: Manage Azure Identities and Governance (15-20%)

## 📚 Learning Objectives

By the end of this module, you will be able to:

- Manage Azure Active Directory (AAD) identities
- Implement and manage Azure RBAC
- Manage subscriptions and tenants
- Configure governance and compliance
- Manage Azure Policy and blueprints

## 📖 Topics Covered

### 1.1 Azure Active Directory (AAD)
- User and group management
- User authentication methods
- Conditional access policies
- Guest user access
- Multi-factor authentication (MFA)

### 1.2 Role-Based Access Control (RBAC)
- Built-in roles
- Custom roles
- Role assignments
- Access reviews
- Privileged Identity Management (PIM)

### 1.3 Subscriptions and Governance
- Subscription management
- Management groups
- Resource tagging
- Cost management
- Azure Advisor

### 1.4 Policies and Compliance
- Azure Policy
- Azure Blueprints
- Regulatory compliance
- Compliance Manager
