# Module 2: Implement and Manage Storage (15-20%)

## 📚 Learning Objectives

By the end of this module, you will be able to:

- Manage storage accounts and properties
- Create and configure Blob Storage
- Create and configure File Shares
- Manage storage security and access
- Monitor storage performance

## 📖 Topics Covered

### 2.1 Storage Account Management
- Storage account types (Standard, Premium)
- Redundancy options (LRS, GRS, RA-GRS, ZRS)
- Storage account properties and settings
- Account access keys and SAS tokens
- Storage account endpoints

### 2.2 Blob Storage
- Blob types (Block, Page, Append)
- Blob containers
- Blob lifecycle management
- Blob snapshots and versioning
- Hot/Cool/Archive tiers

### 2.3 Azure Files
- File shares (SMB, NFS)
- Mounting file shares
- File synchronization (Azure File Sync)
- Share snapshots

### 2.4 Storage Security
- Shared Access Signatures (SAS)
- Storage account keys
- Azure Storage encryption
- Virtual network service endpoints
- Managed identities

### 2.5 Storage Monitoring
- Storage analytics
- Azure Monitor metrics
- Storage insights
- Capacity planning
