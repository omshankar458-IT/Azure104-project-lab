# Module 3: Deploy and Manage Azure Compute Resources (20-25%)

## 📚 Learning Objectives

By the end of this module, you will be able to:

- Deploy and manage virtual machines
- Configure VM extensions and custom script extensions
- Manage VM disks and storage
- Create and manage container instances
- Deploy and manage Azure App Service
- Configure auto-scaling and load balancing

## 📖 Topics Covered

### 3.1 Virtual Machines
- VM creation and sizing
- Image selection (Windows, Linux)
- Disk configuration
- Network configuration
- VM extensions
- Custom script extensions

### 3.2 VM Scaling
- Virtual Machine Scale Sets
- Auto-scaling policies
- Load balancing
- Health probes
- Update and fault domains

### 3.3 Containers
- Azure Container Instances
- Container registries
- Container images
- Environment variables and secrets

### 3.4 App Service
- App Service Plans
- Web apps deployment
- Deployment slots
- Application settings
- Scaling and performance

### 3.5 Azure Kubernetes Service (AKS)
- Cluster creation
- Node pools
- Deployments and services
- Networking and ingress
- Monitoring and logging
