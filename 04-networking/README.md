# Module 4: Configure and Manage Virtual Networking (20-25%)

## 📚 Learning Objectives

By the end of this module, you will be able to:

- Create and configure virtual networks
- Configure subnets and IP addressing
- Implement network segmentation
- Configure network security groups
- Create and manage application gateways
- Configure DNS and load balancing

## 📖 Topics Covered

### 4.1 Virtual Networks
- VNet creation and configuration
- Subnet design
- IP addressing (IPv4, IPv6)
- CIDR notation and subnetting
- VNet peering
- VPN gateways

### 4.2 Network Security
- Network Security Groups (NSG)
- Security rules
- Application Security Groups (ASG)
- Azure Firewall
- DDoS Protection
- Web Application Firewall (WAF)

### 4.3 Load Balancing
- Azure Load Balancer
- Application Gateway
- Traffic Manager
- Front Door

### 4.4 DNS and Connectivity
- Azure DNS
- Private endpoints
- Service endpoints
- ExpressRoute
- VPN connections

### 4.5 Network Monitoring
- Network Watcher
- Connection troubleshooting
- NSG flow logs
- Performance metrics
