# Module 5: Monitor and Maintain Azure Resources (10-15%)

## 📚 Learning Objectives

By the end of this module, you will be able to:

- Configure and manage Azure Monitor
- Create and configure alerts
- Set up diagnostic logging
- Configure application insights
- Create dashboards and workbooks
- Implement backup and disaster recovery

## 📖 Topics Covered

### 5.1 Azure Monitor
- Metrics and logs
- Diagnostic settings
- Log Analytics workspace
- KQL queries
- Custom metrics

### 5.2 Alerting
- Alert rules
- Alert types
- Action groups
- Smart detection
- Notification methods

### 5.3 Application Insights
- Application monitoring
- Performance counters
- Availability tests
- Failure diagnostics
- Custom events and metrics

### 5.4 Dashboards and Reporting
- Azure dashboards
- Workbooks
- Power BI integration
- Custom visualizations

### 5.5 Backup and Disaster Recovery
- Azure Backup
- Recovery Services Vault
- Backup policies
- Disaster recovery
- Business continuity
