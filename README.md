# Azure Administrator Associate (AZ-104) Practice Lab

A comprehensive hands-on practice laboratory for mastering Azure administration skills required for the AZ-104 certification exam.

## 📚 Course Overview

This practice lab covers all domains tested in the Azure Administrator Associate certification:

- **Manage Azure identities and governance** (15-20%)
- **Implement and manage storage** (15-20%)
- **Deploy and manage Azure compute resources** (20-25%)
- **Configure and manage virtual networking** (20-25%)
- **Monitor and maintain Azure resources** (10-15%)

## 🚀 Getting Started

### Prerequisites

- Azure Subscription (Free tier available)
- Azure CLI or PowerShell installed
- Visual Studio Code with Azure extensions
- Basic understanding of cloud computing

### Setup

1. Clone this repository
2. Follow the setup guide in `SETUP.md`
3. Start with Module 1

## 📖 Modules

Each module contains:
- **Overview**: Key concepts and theory
- **Lab exercises**: Step-by-step hands-on tasks
- **Challenge tasks**: Real-world scenarios
- **Review questions**: Self-assessment

## ⭐ Featured Labs

- Identity and Access Management (IAM)
- Storage Accounts and Managed Disks
- Virtual Machines and Scale Sets
- Azure Virtual Networks and Subnets
- Azure Monitor and Log Analytics
